# Emojity
#### Emojis is a module that allows you to quickly transform a word into emoji arrays

## Installation:
`npm i  emojity`
## Importing to your code:
`const { emojity } = require("./emojity");`

### Example usage:
- Creating emoji response to the right word using Discord.js
